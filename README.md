## Partial effect plots with ggplot2

After fitting a GAM model, we are interested in how each factor affects the outcome variable. That’s when we would want to introduce partial effect plots, where we fix all but one independent variable (though this variable can be an interaction of multiple factors), and plot the model predictions against this variable.  

While we do have options to generate PE plots without having to look into any nitty-gritty details, there are times when we wish to gain more customizability. Creating the partial effect plots from scratch is not too complicated, and sometimes it could be easier than trying to bend existing tools to suit our needs. *[Read more](https://longyyu.github.io/static/dataviz-partial-effect.html)*
